## 概要

様々なメモの形態をオールインワンにしたメモアプリ(理想)<br>

1. URL コレクション(実装中)<br>
   ネット上の記事を記録し保存しておくための専用メモ機能

1. シンプルメモ(未実装)<br>
   シンプルなメモ機能

1. 付箋メモ(未実装)
